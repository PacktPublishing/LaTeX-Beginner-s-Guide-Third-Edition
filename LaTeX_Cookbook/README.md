The second, improved, and extended edition of the *LaTeX Cookbook* was
published in March 2024. The first edition (2015) has been revised, and
all code examples have been updated to work with the latest LaTeX
classes and packages (2024).

Check it out here: [LaTeX-Cookbook.net](https://latex-cookbook.net/about)
You will find the code of the book examples together with an online
LaTeX Compiler, reviews, errata, and additional material.

The new edition includes additional examples demonstrating how to:

-   Create fancy word clouds
-   Program with Lua in LaTeX
-   Simulate computer screen output
-   Design puzzles
-   Draw atomic orbitals
-   Visualize chemical elements in the periodic table
-   Create scientific and technical diagrams

[![LaTeX Cookbook
Cover](https://latex.net/wp-content/uploads/2024/03/latex-cookbook-cover.png)](https://latex-cookbook.net/about)

The revised chapters have been rearranged, and a new chapter about using
ChatGPT as a coding assistant and editing tool has been added.

## Contents Overview

### Chapter 1 -- Exploring Various Document Classes

Insight into diverse document types. Learn how LaTeX can create theses,
books, CVs, presentations, flyers, and large posters with tailored
examples.

### Chapter 2 -- Tuning the Text

Customizing text details within documents. Covers fundamentals,
practical tips, and demonstrations of LaTeX's capabilities beyond
standard paragraph formatting.

### Chapter 3 -- Adjusting Fonts

How to make global font choices and adjust fonts locally within your
document.

### Chapter 4 -- Creating Tables

Craft visually appealing tables, align numeric data, use colors, handle
floating tables and figures, merge and split cells, and import table
data from external files.

### Chapter 5 -- Working with Images

Image quality considerations and practical instructions for including,
positioning, shaping, and aligning images in LaTeX.

### Chapter 6 -- Creating Graphics

Step-by-step instructions for creating diagrams and charts using modern
graphics packages.

### Chapter 7 -- Creating Beautiful Designs

Background images, ornaments, attractive headings, calendars, word
clouds, keyboard/menu symbols, and simulated terminal output.

### Chapter 8 -- Producing Contents, Indexes, and Bibliographies

Customize the table of contents, lists of figures and tables,
bibliographies, glossaries, and indexes.

### Chapter 9 -- Optimizing PDF Files

Work with metadata, comments, fillable forms, merge PDFs, adjust
margins, optimize for e-books, and create PDF animations.

### Chapter 10 -- Writing Advanced Mathematics

Advanced mathematical typesetting: refining formulas, creating theorems,
diagrams, geometric figures, and 2D/3D plots.

### Chapter 11 -- Using LaTeX in Science and Technology

Applications in chemistry, physics, computer science, electronics, and
other technical fields.

### Chapter 12 -- Getting Support on the Internet

Guide to valuable LaTeX resources and how to effectively use TeX online
communities.

### Chapter 13 -- Using Artificial Intelligence with LaTeX

Using ChatGPT as a coding assistant and editing tool to enhance
productivity and content quality.
