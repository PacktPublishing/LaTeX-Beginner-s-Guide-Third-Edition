Apart from code examples, this website serves as a place to post improvements, errata, recommendations, and updates, ad you can find reviews by our readers.

The complete code is available on [GitHub](https://github.com/PacktPublishing/LaTeX-graphics-with-TikZ) for browsing, for download as a [ZIP file](/code.zip), and for [opening everything in a single project](https://www.overleaf.com/docs?snip_uri=https://tikz.org/code.zip) on Overleaf.

## What this book covers

- **Chapter 1 – Getting Started with TikZ**  
  Introduction, philosophy, installation, first drawing, documentation tips.

- **Chapter 2 – Creating the First TikZ Images**  
  TikZ syntax, Cartesian and polar coordinates (2D/3D), basic shapes, color.

- **Chapter 3 – Drawing and Positioning Nodes**  
  Nodes, shapes, alignment, labels, images.

- **Chapter 4 – Drawing Edges and Arrows**  
  Connecting nodes, curves, arrow tips, edge labels.

- **Chapter 5 – Using Styles and Pics**  
  Global/local styles, scopes, reusable mini pictures.

- **Chapter 6 – Drawing Trees and Graphs**  
  Trees, mind maps, graphs, matrix layouts.

- **Chapter 7 – Filling, Clipping, and Shading**  
  Path filling, clipping areas, smooth color shading.

- **Chapter 8 – Decorating Paths**  
  Path effects, wavy/zigzag lines, text along paths.

- **Chapter 9 – Using Layers, Overlays, and Transparency**  
  Layers, transparency, overlays, background images.

- **Chapter 10 – Calculating with Coordinates and Paths**  
  Coordinate math, intersections, projections, loops.

- **Chapter 11 – Transforming Coordinates and Canvas**  
  Shifting, rotating, scaling drawings.

- **Chapter 12 – Drawing Smooth Curves**  
  Smooth curve techniques without sharp corners.

- **Chapter 13 – Plotting in 2D and 3D**  
  Axes customization, 2D/3D plots, legends, intersections.

- **Chapter 14 – Drawing Diagrams**  
  Flowcharts, relationship and quantitative diagrams.

- **Chapter 15 – Having Fun with TikZ**  
  Creative examples: animals, flags, shapes, game pieces.
